# Project Documentation

Welcome to the project documentation for the Compute Cluster CLI. This site contains detailed guides and references for using the CLI, managing the cluster, and leveraging our AI-powered development workflows.

Use the navigation to explore different topics, or use the search bar to find specific content.

## Overview

Our 5-node hybrid compute cluster is managed by a custom CLI tool. This documentation covers:

-   **AI Integration**: How we use Aider and Avante to supercharge infrastructure automation with Ansible.
-   **Tool Guides**: In-depth guides for the AI tools we use.
-   **Workflows**: Practical examples of development and operational workflows.
-   **Security**: Best practices for securing our development environment.
